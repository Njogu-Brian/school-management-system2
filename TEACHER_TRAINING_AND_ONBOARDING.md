## Introduction

Welcome to the School Management System! This comprehensive guide will help you understand and effectively use all the features available to teachers. The system is designed to streamline your daily tasks, improve communication with students and parents, and enhance the overall teaching experience.

### What You'll Learn
- How to access and navigate the system
- How to manage attendance, grades, and assessments
- How to communicate with students and parents
- How to plan lessons and manage curriculum
- How to track student behavior and progress

### System Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Internet connection
- Your login credentials (provided by the administrator)

## System Access and Login

### First-Time Login

1. **Access the System**
   - Open your web browser
   - Navigate to the school's system URL (provided by administrator)
   - You'll see the login page

2. **Login Credentials**
   - **Username/Email**: Your work email address
   - **Password**: Initial password provided by administrator
   - Click "Login"

3. **Password Reset (If Required)**
   - If prompted to change password, follow the on-screen instructions
   - Create a strong password (minimum 8 characters, mix of letters, numbers, and symbols)
   - Remember to save your password securely

4. **Dashboard Access**
   - After successful login, you'll be redirected to your Teacher Dashboard
   - The dashboard provides an overview of your classes, students, and tasks

### Navigation Overview

The system uses a sidebar navigation menu with the following main sections:
- **Teacher Dashboard**: Your home page with overview information
- **My Profile**: View and update your personal information
- **Attendance**: Mark and view attendance records
- **Exam Marks**: Enter and manage student grades
- **Report Cards**: View and edit student report cards
- **Homework**: Create and manage homework assignments
- **Digital Diaries**: Communicate with students and parents
- **Student Behaviour**: Record and track student behavior
- **Timetable**: View your teaching schedule
- **CBC Curriculum & Planning**: Access curriculum designs, schemes of work, and lesson plans

## Core Features Training

### 1. Attendance Management

#### Marking Attendance

**Purpose**: Record daily student attendance for your classes.

**Steps**:
1. Navigate to **Attendance** → **Mark Attendance**
2. Select:
   - **Date**: Default is today (can be changed)
   - **Class**: Choose from your assigned classes
   - **Stream** (if applicable): Select specific stream
3. View the student list
4. Mark each student as:
   - **Present** ✓
   - **Absent** ✗
   - **Late** (if applicable)
   - **Excused** (if applicable)
5. Add notes if needed (e.g., reason for absence)
6. Click **Save Attendance**

**Best Practices**:
- Mark attendance at the beginning of each lesson
- Review attendance before saving
- Add notes for unusual absences
- Mark late students appropriately

#### Viewing Attendance Records

**Steps**:
1. Navigate to **Attendance** → **View Records**
2. Use filters:
   - **Date Range**: Select start and end dates
   - **Class**: Filter by specific class
   - **Student**: Search for specific student
3. View attendance summary:
   - Total days present/absent
   - Attendance percentage
   - Absence patterns
4. Export data if needed (Excel/PDF)

**Use Cases**:
- Track attendance trends
- Identify students with frequent absences
- Generate attendance reports for parents
- Prepare attendance data for report cards

### 2. Exam Marks Management

#### Entering Marks

**Purpose**: Record student performance in exams and assessments.

**Steps**:
1. Navigate to **Exam Marks** → **Enter Marks**
2. Select:
   - **Exam**: Choose the exam/assessment
   - **Class**: Select your class
   - **Subject**: Select the subject
   - **Stream** (if applicable)
3. View the student list with entry fields
4. Enter marks for each student:
   - **Score**: Enter the numerical score
   - **Grade**: System may auto-calculate (depending on setup)
   - **Remarks**: Add comments if needed
5. Review entries for accuracy
6. Click **Save Marks**

**Bulk Entry**:
- Use **Bulk Entry** for faster data entry
- Upload Excel file (if supported)
- Copy-paste from spreadsheet

**Best Practices**:
- Enter marks promptly after exams
- Double-check entries before saving
- Use consistent grading scale
- Add remarks for exceptional performance or concerns

#### Viewing Marks

**Steps**:
1. Navigate to **Exam Marks** → **My Class Marks**
2. Filter by:
   - **Exam**: Select specific exam
   - **Class**: Filter by class
   - **Subject**: Filter by subject
3. View:
   - Individual student marks
   - Class average
   - Highest/lowest scores
   - Grade distribution

**Use Cases**:
- Review student performance
- Identify students needing extra support
- Prepare for parent meetings
- Generate performance reports

### 3. Report Cards

#### Viewing Report Cards

**Steps**:
1. Navigate to **Report Cards** → **All Reports**
2. Filter by:
   - **Academic Year**: Select year
   - **Term**: Select term
   - **Class**: Filter by class
3. Click on a student's report card to view details

#### Editing Skills and Remarks

**Purpose**: Add skills assessment and remarks for students.

**Steps**:
1. Navigate to **Report Cards** → **Skills Editor**
2. Select the report card period
3. For each student:
   - **Skills**: Rate skills (e.g., Communication, Critical Thinking)
   - **Remarks**: Write general remarks
   - **Subject Remarks**: Add subject-specific comments
4. Save changes

**Best Practices**:
- Be specific and constructive in remarks
- Highlight strengths and areas for improvement
- Use professional language
- Complete all required fields

---

### 4. Homework Management

#### Creating Homework

**Purpose**: Assign homework to students.

**Steps**:
1. Navigate to **Homework** → **Create Homework**
2. Fill in details:
   - **Title**: Descriptive title
   - **Subject**: Select subject
   - **Class**: Select class(es)
   - **Due Date**: Set deadline
   - **Instructions**: Detailed assignment instructions
   - **Attachments**: Upload files if needed (PDF, Word, etc.)
3. Set options:
   - **Allow Late Submission**: Yes/No
   - **Maximum Score**: Total points
4. Click **Create Homework**

**Best Practices**:
- Provide clear, detailed instructions
- Set realistic due dates
- Attach resources if needed
- Review before publishing

#### Reviewing Homework Submissions

**Steps**:
1. Navigate to **Homework**
2. Click on a homework assignment
3. View submissions:
   - **Submitted**: Students who submitted
   - **Pending**: Students who haven't submitted
   - **Late**: Submissions after due date
4. Review each submission:
   - Open student work
   - Grade the assignment
   - Add feedback/comments
   - Mark as reviewed

**Grading**:
- Enter score out of maximum
- Add written feedback
- Mark as complete
- Notify students of grades

## Daily Workflow

### Morning Routine

1. **Check Dashboard**
   - Review pending tasks
   - Check announcements
   - Review today's schedule

2. **Mark Attendance**
   - Mark attendance for first period
   - Continue for subsequent periods

3. **Review Homework**
   - Check new submissions
   - Grade completed assignments

### During Lessons

1. **Use Digital Diary**
   - Post lesson updates or reminders for a specific student, an entire class, or the whole school (admins only)
   - Attach supporting files (photos, PDFs, etc.)
   - Review replies from parents and other staff in the threaded view

#### Posting a Diary Entry (Teacher/Admin)
1. Open **CBC Curriculum & Planning → Digital Diaries**.
2. Use the **New Entry** button.
3. Select a **Target**:
   - **Specific Student** – direct message between staff and one learner + parent.
   - **Classroom** – sends the same note to all students assigned to that class.
   - **Entire School** – only Super Admin/Admin can broadcast to everyone.
4. Type the update, attach optional files, and submit. Every diary keeps a running history so you can scroll back through the full conversation.

#### Responding to a Student’s Diary
1. From the diary list, click **Open** on a student.
2. Review the full timeline, which shows who authored each entry (teacher/admin/parent) with timestamps.
3. Add your response at the bottom; parents will see it instantly in their portal.

#### Parent Experience (for onboarding parents)
1. Parents access **Parent Portal → Diaries** to see each child’s feed.
2. They can read teacher/admin notes and post their own updates or attachments.
3. Entries automatically mark as read so staff can track engagement.

2. **Record Behavior**
   - Note positive behaviors
   - Document concerns

### End of Day

1. **Complete Attendance**
   - Ensure all classes marked
   - Review attendance data

2. **Enter Marks** (if applicable)
   - Enter any assessments completed
   - Review and save

3. **Plan for Tomorrow**
   - Review next day's schedule
   - Prepare materials
   - Set reminders

## Troubleshooting

### Common Issues

#### Login Problems

**Issue**: Cannot log in
- **Solution**: 
  - Check username/email spelling
  - Verify password (check caps lock)
  - Contact administrator if password forgotten
  - Clear browser cache

#### Attendance Not Saving

**Issue**: Attendance marks not saving
- **Solution**:
  - Check internet connection
  - Verify date is correct
  - Ensure all required fields filled
  - Try refreshing page
  - Contact IT support if persists

#### Marks Entry Errors

**Issue**: Cannot enter marks
- **Solution**:
  - Verify exam is open for entry
  - Check you have permission
  - Ensure correct class/subject selected
  - Contact administrator if needed

#### Slow Performance

**Issue**: System running slowly
- **Solution**:
  - Check internet connection
  - Close unnecessary browser tabs
  - Clear browser cache
  - Try different browser
  - Contact IT support
